6.Print Prime Numbers in an Array (Company Specific)
Problem Statement
Given an array of N integers, print all the prime numbers present in the array.
If the array does not contain any prime numbers, print No Prime Numbers.
A prime number is a positive integer greater than 1 that has exactly two factors: 1 and itself.
Input Format
N
N integers
Output Format
Print all prime numbers separated by a space.

If no prime numbers exist, print:-1
Constraints
1 ≤ N ≤ 100
0 ≤ Array Element ≤ 100000
Sample Input 1
6
10 17 21 29 35 41
Sample Output 1
17 29 41

Test Cases
Test Case 1
Input
6
10 17 21 29 35 41

Output
17 29 41

Test Case 2
Input
5
4 6 8 9 10
Output
-1
